# 预测模型

**输入：一段操作序列**

```
[
    [操作向量],
    [操作向量],
    [操作向量],
    …
]
```

**输出：预测的一个操作**

```
[预测操作向量]
```



# 模式识别模型

**输入：一段操作序列**

```
[
    [操作向量],
    [操作向量],
    [操作向量],
    …
]
```

**输出：匹配到了哪个模式**

```
数值
```



# 操作向量

```
[
	操作类型, ([操作件], [操作件的关联项])
]

操作类型：event_type
操作件：[artifact1, artifact2, artfact3...]
关联项：[[操作件的父件], [调用了操作件的件], [操作件调用的件]]
```

# 预测操作向量

```
[操作类型，操作件]
```



操作序列-预处理-编码-操作向量-预测-预测操作向量-解码-预测操作
